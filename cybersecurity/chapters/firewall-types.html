Typy firewallů a jejich funkce

Firewally podle vrstev OSI

Network Layer Firewall

  • Filtrace na základě IP adres
  • Kontrola síťové vrstvy protokolu
  • Příklad: Filtrování podle zdrojové IP

Transport Layer Firewall

  • Kontrola TCP/UDP spojení
  • Stavová inspekce paketů
  • Příklad: Filtrace podle portů

Application Layer Firewall

  • Analýza aplikačních protokolů
  • Hluboká inspekce paketů
  • Příklad: Filtrace HTTP požadavků

Specializované firewally

Context Aware Firewall

  • Rozhodování podle kontextu komunikace
  • Adaptivní změna pravidel
  • Příklad: Analýza chování uživatele

Host-based Firewall

  • Ochrana konkrétního zařízení
  • Lokální filtrace provozu
  • Příklad: Windows Defender Firewall

NAT Firewall

  • Překládání síťových adres
  • Skrývání vnitřní struktury sítě
  • Příklad: Překlad privátních adres

Proxy servery

Forward Proxy

  • Zprostředkování přístupu do internetu
  • Cachování obsahu pro klienty
  • Příklad: Firemní proxy server

Reverse Proxy

  • Ochrana vnitřních webových serverů
  • Rozložení zátěže provozu
  • Příklad: Nginx jako reverse proxy

Application Proxy

  • Specializované proxy pro aplikace
  • Hluboká analýza protokolů
  • Příklad: SMTP proxy pro email

Pokročilé funkce

Deep Packet Inspection

  • Detailní analýza obsahu paketů
  • Identifikace škodlivého obsahu
  • Příklad: Antivirová kontrola provozu

Virtual Firewall

  • Virtualizované bezpečnostní funkce
  • Oddělení síťových segmentů
  • Příklad: VMware NSX firewall

Next-Gen Firewall

  • Kombinace více bezpečnostních funkcí
  • Inteligentní analýza hrozeb
  • Příklad: Palo Alto Networks
