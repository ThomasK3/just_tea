Šifrování a kryptografické metody

Základní typy šifrování

Symetrická kryptografie

  • Stejný klíč pro šifrování/dešifrování
  • Rychlé zpracování velkých objemů dat
  • Příklad: AES-256 pro šifrování disku

Asymetrická kryptografie

  • Pár veřejného a soukromého klíče
  • Bezpečnější distribuce klíčů
  • Příklad: RSA pro digitální podpisy

Hybridní šifrování

  • Kombinace symetrického a asymetrického
  • Využívá výhody obou přístupů
  • Příklad: HTTPS komunikace

Hashování a podpisy

Hashovací funkce

  • Jednosměrná transformace vstupních dat
  • Vytváří unikátní otisk dat
  • Příklad: SHA-256 pro kontrolu integrity

Digitální podpis

  • Kryptografické ověření původu dat
  • Kombinace hashování a asymetrického šifrování
  • Příklad: Podepisování dokumentů PDF

Message Authentication Code

  • Ověření integrity a autenticity
  • Kombinace hashe a sdíleného klíče
  • Příklad: HMAC pro API autentizaci

Infrastruktura veřejných klíčů

Certifikační autorita

  • Vydávání digitálních certifikátů
  • Ověření identity žadatelů
  • Příklad: Let's Encrypt pro HTTPS

Digitální certifikáty

  • Vazba veřejného klíče na identitu
  • Obsahuje metadata a podpis CA
  • Příklad: X.509 certifikát serveru

Revokace certifikátů

  • Zneplatnění kompromitovaných certifikátů
  • Správa CRL a OCSP
  • Příklad: Blacklist neplatných certifikátů

Bezpečné připojení

VPN (Virtual Private Network)

  • Šifrovaný tunel přes veřejnou síť
  • Zajištění důvěrnosti přenosu dat
  • Příklad: OpenVPN pro vzdálený přístup

SSL/TLS

  • Protokol pro zabezpečení komunikace
  • Handshake pro ustanovení klíčů
  • Příklad: HTTPS zabezpečení webu

IPsec

  • Zabezpečení na síťové vrstvě
  • Šifrování IP paketů
  • Příklad: Site-to-site VPN tunel

Praktická implementace

Správa klíčů

  • Bezpečné generování a úschova
  • Pravidelná rotace klíčů
  • Příklad: HSM pro kritické klíče

Kryptografické protokoly

  • Standardizované bezpečné postupy
  • Ověřené implementace knihoven
  • Příklad: TLS 1.3 konfigurace

Best practices

  • Aktuální kryptografické standardy
  • Pravidelný bezpečnostní audit
  • Příklad: Použití Forward Secrecy
