Aplikační vrstva TCP/IP

Modely komunikace

Peer-to-peer (P2P)

  • Přímá komunikace mezi rovnocennými uzly
  • Decentralizovaná architektura bez serveru
  • Používán pro sdílení souborů (BitTorrent)

Klient-server

  • Hierarchický model síťové komunikace
  • Server poskytuje služby klientům
  • Používán pro webové služby

Základní protokoly

HTTP/HTTPS

  • Protokol pro přenos webových stránek
  • Používá porty 80 a 443
  • Zajišťuje zabezpečený přenos dat

FTP

  • Protokol pro přenos souborů
  • Používá porty 20 a 21
  • Umožňuje nahrávání/stahování souborů serveru

SMTP

  • Protokol pro odesílání emailů
  • Používá port 25
  • Zajišťuje doručování pošty servery

Další protokoly

SSH

  • Zabezpečený vzdálený přístup k systémům
  • Používá port 22
  • Nahrazuje nezabezpečený Telnet protokol

NTP

  • Synchronizace času v síti
  • Používá port 123
  • Zajišťuje přesný čas systémů

DNS

  • Překlad doménových jmen na IP
  • Používá port 53
  • Hierarchický systém doménových jmen

Sdílení souborů

SMB/CIFS

  • Protokol pro sdílení souborů Windows
  • Používá port 445
  • Umožňuje přístup ke sdíleným adresářům

NFS

  • Síťový souborový systém Unix/Linux
  • Používá port 2049
  • Poskytuje transparentní přístup k souborům
