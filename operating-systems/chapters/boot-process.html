Start počítače a bootovací proces

Základní fáze startu

Reset a inicializace

  • Aktivace signálu reset pro všechny obvody
  • Uvedení hardwaru do výchozího stavu
  • Zahájení činnosti v 16bitovém režimu CPU

BIOS/UEFI

  • Základní programové vybavení na základní desce
  • Poskytuje služby pro práci s hardwarem
  • Zodpovídá za počáteční inicializaci systému

Bootovací proces

POST test

  • Power On Self Test kontrola hardwaru
  • Detekce připojených zařízení a chyb
  • Zobrazení základních informací o systému

Boot sektor

  • První sektor bootovacího zařízení (512B)
  • Obsahuje kód pro zavedení operačního systému
  • Na disku součást MBR nebo GPT

Typy bootování

Start z disku

  • Načtení MBR/GPT z prvního sektoru
  • Výběr aktivního oddílu pro boot
  • Spuštění bootloaderu operačního systému

Start z jiných médií

  • Boot z CD/DVD pomocí ISO9660
  • Boot z USB pomocí USB Mass Storage
  • Síťový boot přes PXE protokol

Moderní technologie

Secure Boot

  • Ověření digitálního podpisu bootloaderu
  • Ochrana proti neautorizovanému kódu
  • Součást UEFI specifikace

UEFI

  • Nástupce klasického BIOSu
  • Podporuje GPT a větší disky
  • Nabízí pokročilé bootovací možnosti
