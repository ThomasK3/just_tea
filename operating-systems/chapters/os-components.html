Operační systém a jeho součásti

Funkce operačního systému

Ovládání počítače

  • Zajišťuje základní řízení hardwaru počítače
  • Umožňuje spouštění a ukončování programů
  • Zajišťuje přenos dat mezi I/O zařízeními

Abstrakce hardware

  • Poskytuje jednotné rozhraní pro programy
  • Využívá API pro komunikaci s hardwarem
  • Odděluje aplikace od fyzického hardware

Správa prostředků

  • Přiděluje systémové prostředky jednotlivým procesům
  • Řídí přístup k sdíleným komponentám
  • Kontroluje využití CPU, paměti, I/O

Typy jader OS

Monolitické jádro

  • Všechny funkce jsou v jednom celku
  • Běží v privilegovaném režimu procesoru
  • Příklad: Linux, FreeBSD

Mikrojádro

  • Obsahuje pouze nejnutnější funkce systému
  • Služby běží jako samostatné procesy
  • Zaměřeno na modularitu a bezpečnost

Hybridní jádro

  • Kombinuje vlastnosti obou předchozích typů
  • Kompromis mezi výkonem a modularitou
  • Příklad: Windows NT

RTOS (Real-Time OS)

Hard RTOS

  • Garantuje pevné časové limity odezvy
  • Určen pro kritické řídící systémy
  • Využití v průmyslu a embedded systémech

Soft RTOS

  • Flexibilnější časové limity než Hard RTOS
  • Toleruje občasné překročení limitů
  • Využíván v multimediálních aplikacích

Multiprocessing

Symetrický (SMP)

  • Všechny procesory jsou rovnocenné
  • Sdílí společnou paměť a periferie
  • Nejčastější architektura současných systémů

Asymetrický (ASMP)

  • Procesory mají různé role
  • Dedikované procesory pro specifické úlohy
  • Využití ve specializovaných systémech
