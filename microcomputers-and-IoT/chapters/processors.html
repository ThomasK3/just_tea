Procesory

1. Architektura procesorů

Von Neumannova architektura

  • Společná paměť pro program/data
  • Sekvenční zpracování instrukcí
  • Základní model většiny CPU

Harvardská architektura

  • Oddělená paměť program/data
  • Paralelní přístup k instrukcím/datům
  • Typická pro mikrokontroléry

RISC vs CISC

  • Rozdílná filozofie instrukční sady
  • RISC má jednodušší instrukce
  • CISC má komplexnější instrukce

2. Komponenty CPU

ALU (Aritmeticko-logická jednotka)

  • Provádí matematické a logické operace
  • Zpracování celých čísel
  • Základní výpočetní jednotka procesoru

Řadič

  • Řídí tok instrukcí procesoru
  • Dekóduje instrukce
  • Koordinuje činnost komponent

Registry

  • Rychlá pracovní paměť CPU
  • Ukládání mezivýsledků
  • Řízení stavu procesoru

3. Výkonnostní prvky

Pipeline

  • Zřetězené zpracování instrukcí
  • Paralelní vykonávání různých fází
  • Zvýšení výkonu procesoru

Cache paměť

  • Rychlá vyrovnávací paměť
  • Hierarchické uspořádání (L1/L2/L3)
  • Snížení latence přístupu

Branch prediction

  • Predikce větvení programu
  • Spekulativní vykonávání instrukcí
  • Optimalizace pipeline

4. Moderní technologie

Multithreading

  • Současné zpracování více vláken
  • Efektivní využití pipeline
  • Zvýšení propustnosti CPU

Multicore

  • Více výpočetních jader
  • Paralelní zpracování úloh
  • Škálování výkonu

Virtualizace

  • Hardwarová podpora virtualizace
  • Izolace virtuálních strojů
  • Efektivní přepínání kontextů
