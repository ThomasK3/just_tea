Internet věcí (IoT)

1. Základní koncepty

IoT Architektura

  • Propojení fyzických zařízení s internetem
  • Třívrstvý model (věc-síť-cloud)
  • Sběr dat ze senzorů domácnosti

Koncová zařízení

  • Hardwarové jednotky s připojením
  • Obsahují senzory a aktuátory
  • Měření teploty, vlhkosti, spotřeby

Edge Computing

  • Zpracování dat blízko zdroje
  • Redukce přenosu do cloudu
  • Filtrace a agregace senzorových dat

2. Komunikační protokoly

MQTT

  • Lightweight messaging protokol
  • Publish-subscribe komunikační model
  • Přenos dat ze senzorů

CoAP

  • REST protokol pro omezená zařízení
  • Optimalizovaný pro IoT aplikace
  • Komunikace mezi IoT zařízeními

HTTP/HTTPS

  • Standardní webový protokol
  • REST API pro IoT služby
  • Připojení do cloudových platforem

3. Bezpečnost

Autentizace zařízení

  • Ověření identity IoT zařízení
  • Použití certifikátů a tokenů
  • Zabezpečení přístupu do sítě

Šifrování dat

  • Ochrana přenášených dat
  • Použití TLS/SSL protokolů
  • Zabezpečení citlivých informací

Aktualizace firmware

  • Vzdálená aktualizace zařízení
  • Oprava bezpečnostních chyb
  • Over-the-air (OTA) aktualizace

4. IoT Platformy

Cloud služby

  • Centralizovaná správa IoT zařízení
  • Škálovatelné úložiště dat
  • AWS IoT, Azure IoT Hub

Vizualizace dat

  • Dashboardy pro monitoring zařízení
  • Grafické zobrazení měřených hodnot
  • Grafana, Thingsboard, Kibana

Automatizace

  • Pravidla pro zpracování událostí
  • Reakce na měřené hodnoty
  • Automatické řízení domácnosti
