Paměti

1. Základní rozdělení

Volatilní paměti

  • Ztrácí obsah při odpojení napájení
  • Operační paměť RAM
  • Registry a cache CPU

Nevolatilní paměti

  • Uchovávají data bez napájení
  • Flash, EEPROM, ROM
  • Dlouhodobé uložení dat

Cache paměti

  • Rychlá vyrovnávací paměť
  • Hierarchické uspořádání L1-L3
  • Snížení přístupové doby

2. RAM technologie

DRAM

  • Dynamická paměť s refresh
  • Vysoká hustota dat
  • Hlavní operační paměť

SRAM

  • Statická paměť bez refresh
  • Rychlejší než DRAM
  • Použití v cache CPU

DDR SDRAM

  • Synchronní dynamická RAM
  • Double Data Rate přenos
  • Různé generace (DDR4/DDR5)

3. Flash paměti

NOR Flash

  • Přímé vykonávání kódu
  • Náhodný přístup
  • Boot ROM aplikace

NAND Flash

  • Vysoká hustota dat
  • Blokový přístup
  • SSD disky, USB flash

eMMC/UFS

  • Embedded paměťové systémy
  • Integrovaný řadič
  • Mobilní zařízení

4. Speciální paměti

NVRAM

  • Nonvolatilní RAM
  • Záloha konfigurace
  • CMOS nastavení

MRAM

  • Magnetická RAM
  • Neomezený počet zápisů
  • Průmyslové aplikace

ReRAM

  • Resistive RAM
  • Změna odporu materiálu
  • Budoucí paměťová technologie

5. Paměťové systémy

ECC paměti

  • Detekce a oprava chyb
  • Serverové systémy
  • Zvýšená spolehlivost

Virtuální paměť

  • Rozšíření RAM na disk
  • Stránkování paměti
  • Správa paměti OS

Paměťové řadiče

  • Řízení přístupu k paměti
  • Timing a refresh
  • Integrované v CPU/chipset
